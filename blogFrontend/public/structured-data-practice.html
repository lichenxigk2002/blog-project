测试方法

  2. 访问 Google测试工具
  3. 粘贴URL并测试
  4. 查看结构化数据是否正确识别

常见属性说明

属性 说明 示例
@context 指定Schema.org标准 "https://schema.org"
@type 内容类型 "Article", "Product", "Person"
headline 标题 "文章标题"
description 描述 "文章描述"
author 作者 {"@type": "Person", "name": "作者名"}
datePublished 发布日期 "2024-01-15"
image 图片 {"@type": "ImageObject", "url": "图片URL"}
