Meta标签测试页面

这个页面用于测试各种meta标签是否正确设置。

测试步骤：

  1. 右键 -> 查看页面源代码
  2. 搜索 "meta" 查看meta标签
  3. 按F12打开开发者工具
  4. 在Console中运行测试代码

Console测试代码：

// 测试description
console.log('Description:', document.querySelector('meta[name="description"]')?.content);

// 测试Open Graph
console.log('OG Title:', document.querySelector('meta[property="og:title"]')?.content);
console.log('OG Description:', document.querySelector('meta[property="og:description"]')?.content);

// 测试Twitter Card
console.log('Twitter Card:', document.querySelector('meta[name="twitter:card"]')?.content);

// 测试JSON-LD
const jsonLd = document.querySelector('script[type="application/ld+json"]');
if (jsonLd) {
    console.log('JSON-LD:', JSON.parse(jsonLd.textContent));
} else {
    console.log('未找到JSON-LD');
}
